TowerJob¶

class games.necrowar.tower_job.TowerJob¶

Bases: games.necrowar.game_object.GameObject

The class representing the TowerJob in the Necrowar game.

Information about a tower’s job/type.

property all_units¶

Whether this tower type hits all of the units on a tile (True) or one at a time (False).

Type

bool

property damage¶

The amount of damage this type does per attack.

Type

int

property game_object_name¶

String representing the top level Class that this game object is an instance of. Used for reflection to create new instances on clients, but exposed for convenience should AIs want this data.

Type

str

property gold_cost¶

How much does this type cost in gold.

Type

int

property health¶

The amount of starting health this type has.

Type

int

property id¶

A unique id for each instance of a GameObject or a sub class. Used for client and server communication. Should never change value after being set.

Type

str

log(message: str) → None¶

Adds a message to this GameObject’s logs. Intended for your own debugging purposes, as strings stored here are saved in the gamelog.

Parameters

message (str) – A string to add to this GameObject’s log. Intended for debugging.

property logs¶

Any strings logged will be stored here. Intended for debugging.

Type

list[str]

property mana_cost¶

How much does this type cost in mana.

Type

int

property range¶

The number of tiles this type can attack from.

Type

int

property title¶

The type title. ‘arrow’, ‘aoe’, ‘ballista’, ‘cleansing’, or ‘castle’.

Type

‘arrow’, ‘aoe’, ‘ballista’, ‘cleansing’, or ‘castle’

property turns_between_attacks¶

How many turns have to take place between this type’s attacks.

Type

int
